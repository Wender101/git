Comandos do git

  1. Geralmente usando no inicio do projeto 
        //* Serve para iniciar um novo projeto
    git init
  2. Usado para ver o status 
        //* Serve para ver o status dos arquivos da pasta
    git status
  3. Usando para adicionar no fluxo do git 
        //? Usando para adicionar no fluxo do git, dessa forma será possivél salvar as alterações feitas nele dps.
    git add .\'nome do arquivo'

    //? Caso queira adicionar todos os arquivos de uma só vez usa-se o comando
    git add .
  4. Usado para "comentar" uma alteração 
        //; Com o comando:
    git commit -a -m "comentario dentro das aspas duplas"
  5. Usado para adicionar um repositorio remoto 
        //? Para adicionar repositorio um remoto, usaremos o comando:
    git remote add origin 'link do repositorio, OBS: sem as aspas'
  6. Usado para remover um repositorio remoto 
        //! Para remover repositorio remoto, usaremos o comando:
    git remote remove origin
  7. Usado para parear o main 
        //? Para parear o main, usaremos o comando:
    git branch -M main
  8. Usado para enviar os arquivos locais para o repositorio 
        //? Para eviar os arquivos locais para o repositorio, usaremos o comando:
    git push -u origin main
  9. Usado para criar um novo branch, com isso você podera programar em um branch diferente dos outros 
        //; Usado para criar um novo branch
    git checkout -b "nome do novo branch, dentro das aspas duplas"
  10. Enviar novas alterações para o repositorio 
        //! Usado para mandar novas alterações para o repositorio
    git push
  11. Entrar em um branch 
        //; Usado para entrar em um branch
    git checkout 'nome do branch OBS: sem as aspas'
  12. Mostra todos os branchs da aplicação 
        //? Usado para mostra todos os branchs da aplicação
    git branch
  13. Vai unir o branch desejado com o branch que o usuario está. Este comando é geralmente usado após o '11º' comando da lista 
        //! Usado para mandar novas alterações para o repositorio
    git merge 'nome do branch que queira unir ao branch que você está. OBS: sem as aspas'
